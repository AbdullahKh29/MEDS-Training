# Lecture 1: Introduction to Computer Architecture
* **The Computing Stack**: Understanding the "Transformation Hierarchy" from Problems -> Algorithms -> OS -> ISA -> Microarchitecture -> Logic -> Circuits -> Electrons.
* **The von Neumann Model**: Introduction to the fundamental components: Memory, Control Unit, ALU, and I/O.
* **The Architect's Goal**: Crossing abstraction layers to find global optimizations that a single-layer designer would miss.
