# Lecture 2: MOS Transistors and CMOS Logic
* **MOSFETs as Switches**: NMOS (conducts when Gate is High) and PMOS (conducts when Gate is Low).
* **CMOS Complementary Design**: Using both types to ensure zero static power consumption when the circuit is idle.
* **Logic Gate Construction**: Mapping Pull-Up Networks (PUN) using PMOS and Pull-Down Networks (PDN) using NMOS to create NAND and NOR gates.
