# Lecture 3: Combinational Logic & Arithmetic
* **Multiplexers (MUX)**: The 2-to-1 MUX as the "universal" combinational building block ('A + SB$).
* **Full Adder Logic**: Implementing the Sum ( \oplus B \oplus C_{in}$) and Carry Out ( + BC_{in} + AC_{in}$) equations.
* **Ripple Carry Adder (RCA)**: Analysis of the "Critical Path"—the longest delay from the least significant bit's carry-in to the most significant bit's carry-out.
