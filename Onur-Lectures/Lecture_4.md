# Lecture 4: Sequential Logic Fundamentals
* **Bistability**: How cross-coupled inverters create a stable state to store a single bit.
* **Latches vs. Flip-Flops**: 
    * **D-Latch**: Level-sensitive (transparent when clock is high).
    * **D-Flip-Flop**: Edge-triggered (captures data only on the rising/falling edge).
* **The Clock**: Providing a periodic signal to define "discrete time" for the system.
